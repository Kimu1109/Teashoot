<Q&Aコーナー>

Q操作はどうすればいいですか?
Aキーボード.pngを使ってください。

Qボスを倒したと思ったら全然ステージが進みません、どうしたらいいですか?
A[F3]キーを押して[DEBUG?SCROLLON]と入力してエンターキーを押してください。
　それでもダメな場合はもう一度[F3}キーを押して[DEBUG?MAPRELOAD]と入力してエンターキーを押してください。
　この技はステージ3以外で実行してください。

Q[!Sprait Over!]というエラーが出てきます。一体どうしたらいいですか?
Aこれは処理できるスプライトの数以上にスプライトが出てきたときに発生します。
　これを対処するには[F3]キーをおして[DEBUG?HR=1000]を入力してエンターを押してください。
　これを実行するとスプライトの最大数が増えるのでその分処理が多くなり、重くなる場合がほとんどです。
　それでも良ければ実行してください。この設定は次回起動する際にはリセットされます。

Q[F8]キーが効きません。どうすればいいですか?
A長押しするとON、OFFが何回も起きて良くないことが起きます。
　なのでゆっくりポチ!と押してやってください。

Q[FPS]が安定しません。どうすればいいですか?
A[FPS]表示でFPSを確認している場合はステージ1以降信じてはいけません。
　ちゃんとしたFPSを計測したい場合は専用アプリなどを使用してください。
